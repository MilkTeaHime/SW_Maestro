	몰카탐지기가 이미 나와있지만 별도의 커다란 기기를 구입해야 한다.
		// 1. LED
		// 2. 적외선인지 가시광선인지
 - 개념
	핸드폰 케이스 등으로 빛을 쏘아 카메라렌즈에 반사되는 빛을 잡아낸다.
	스마트폰 카메라를 통해 입력된 영상으로부터 카메라렌즈로 의심되는 부분을 CV를 통해 탐지한다.	// 탐지율 <- 단순한 모양에 확실한 색 대비이므로 낮진 않을 것 // 좀 더 깊이 알아볼 것.
	신고 또는 공유 기능을 이용해 지도에 현황을 표시한다.	// 
 - 사업화방향
	1-1. 어느 정도 이상 높은 탐지율을 확보하여 소비자 신뢰를 얻는다. (언론 등을 최대한 활용)
	1-2. 케이스 디자인을 예쁘게 하여 널리 퍼뜨린다.
	2-1. 모텔 업주에게 클린존 정보표시 등을 제안하여 대량구매를 유도한다.
	2-2. 야놀자 등에 협약하여 모텔 업주 등에게 대여/배포한다.
 - 기대효과
	검거율 상승으로 인한 범죄예방
 - 기술성
	CV	// 라이브러리, API, 사용언어 근거 포함하여 조사
 - 추가구현
	위험지역 push 알림 : 빈도가 매우 높은 지역 진입시 푸시. 계속 관심을 갖고 이용하게 만드는 장치.
 - 개발이슈
	핸드폰 케이스 충전 문제 : 케이스가 불룩해진다.
	가격 상승 문제 : LED 자체를 디자인이나 다른 부분과 연결시키는 부분 고려.
	충격에 의한 케이스 고장 문제 : 케이스가 본래의 역할을 수행하지 못한다.
	이미 있는 유사앱 : 아이폰만 나와있는데 특허 준비 중. 
	거울 뒤를 판단할 수 없다 : 자기장센서 및 매직미러 판별법으로 보완
	반짝이는 것이라면 모두 렌즈로 탐지한다 : CV를 좀 더 정교하게 (1. 원만 인식, 2. 카메라 수준 크기만 인식, 3. 의심위치 표시)
	나사까지 탐지해버린다는데...
